# 🔥 Flaymr — AI Code Reviewer

> Paste your code. We'll destroy it. Then fix it.

Flaymr is a brutally honest AI-powered code reviewer that analyzes your code for bugs, security vulnerabilities, performance issues, and best practices — then gives you the fixed version.

## ✨ Features

- 🔥 **Brutal AI Roasts** — Honest, funny, and helpful code reviews
- 📊 **Score Breakdown** — Security, Performance, Readability, Best Practices
- 🐛 **Issue Detection** — Critical, Major, Minor issues with explanations
- ✅ **Fixed Code** — AI rewrites your code the right way
- 🕐 **Roast History** — Track all your past reviews
- 🌗 **Dark/Light Mode** — Easy on the eyes
- 📱 **Fully Responsive** — Works on all devices

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite |
| Styling | Tailwind CSS |
| Routing | React Router |
| AI | Groq API (Llama 3.3 70B) |
| Deploy | Vercel |

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/Flaymr.git
cd Flaymr
```

### 2. Install dependencies
```bash
cd frontend
npm install
```

### 3. Add your API key
```bash
# Create .env file in frontend folder
VITE_GROQ_API_KEY=your_groq_api_key_here
```

### 4. Run locally
```bash
npm run dev
```
